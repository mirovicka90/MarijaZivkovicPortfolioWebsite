My Recent Project

Memory Game

Implemented the application using JQuery, JavaScript, HTML, and CSS. The application consists of a configuration page where users can enter the names of two players, a dropdown list for selecting a duration of the time available for each player moves, and a radio button to select the level of complexity. The first level has 6 pairs of images, the second one 12, and the third one 16. The main application goal is to find 2 same images. The player who finds a higher number of the image pairs, for the configured time, is the winner. Also, the player's scores and the number of attempts are counted and displayed. In the end, the winner is announced.
